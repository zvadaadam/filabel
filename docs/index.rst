Filabel's documentation!
========================

:py:mod:`filabel` is a tool for updating GitHub pull requests (PRs) labels based on the changed files.

It works by providing configuration file, where we define relationship between labels and filenames.
Thanks to the :py:mod:`filabel`, your pull requests can be instantly labeled by our define structure.

:py:mod:`filabel` either can be used as command line tool or like a web application.
